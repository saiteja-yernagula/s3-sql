datatypes
1) varchar(size)
it is used to create variable length string

syntax  
name varchar(20)
email varchar(30)
city varchar(20)

varchar wont occupy space for empty slots


2) char(size)
it is used to create fixed length string

syntax
pincode char(6)
phno char(10)

it will ocupy space even it is having empty slots

3) int
it is used to store the whole numbers (negative and postive)

syntax
id int
age int
price int

with a precision and scale

18.46

scale-2

ddl-data definiton language
it is used define and manage the structure  of
the table,database ,users

1) create
it is used to create the database,table,user 

2) show 
it is used to list out the databases and tables

3) desc
it is used to show the description of table -like 
tatal columns and  datatypes and rules

4) alter & rename
it is used to modify the structure of a table

5) drop 
it is used to remove database and tables 

6) truncate
it is used to clear the values in a table but not
a strcture 


1) create
syntax
CREATE TABLE tbname (col datatype,col2 datatype);

ex 
create table students (id int,name varchar(20));

ex

drop a column
